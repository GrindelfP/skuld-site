Skuld is a Norn (a fate goddess) from Scandinavian mythology. My choice when picking her name as a title for my project as well as (probably) my numerical integration library is dictated by the nature of the goddess: she foretells what should happen the same as a neural network foretells the function for inputted parameters, not calculates it. Also, the reason is that I like Scandinavian mythology very much.

Description

My Bachelor's degree diploma thesis. The main goal is to implement a numerical integration library for neural network-based numerical integration of complex dimensional functions.
